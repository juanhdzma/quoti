# Quoti — Brand Identity

## Concept

Quoti is a quotes app with a journal soul. The experience should feel like opening a well-kept notebook — warm, personal, intentional — but built with the clarity and precision of modern iOS design.

## Voice & tone

- Calm and thoughtful, never loud.
- Personal but not sentimental.
- Minimal copy — let the quotes breathe.

## Visual language

**Feel:** Editorial notebook. Clean paper, ink contrast, a single warm accent.

**Not:** Vintage scrapbook, cold minimalism, flashy gradients.

**Key tension to maintain:** warm materials + modern structure. The warmth comes from the color palette and typography; the modernity comes from spacing, hierarchy, and restraint.

## Typography

| Role           | Font            | Style              |
|----------------|-----------------|--------------------|
| Quotes / body  | Lora            | Italic, 400–600    |
| UI / labels    | Inter           | Regular–Semibold   |
| Display titles | Lora            | Semibold           |

- Lora carries the editorial, journal character.
- Inter keeps the UI clean and legible at small sizes.
- Never use more than these two families.

## Iconography

- Line icons, 1.8px stroke, rounded caps.
- No filled icons except when active state requires it.
- Keep icons simple — they support navigation, not decoration.

## Color philosophy

Single accent. The terracotta (`#C4604A`) is the only color with personality — everything else is neutral. This keeps the quotes as the visual focal point.

See [colors.md](colors.md) for full palette and usage rules.

## Layout principles

- Generous padding — content needs room.
- Left-aligned text everywhere except centered empty states.
- Cards with `24px` corner radius for featured content, `16px` for secondary.
- Subtle separators (`ruled`) over heavy borders.

## What Quoti is not

- Not a productivity app — no streaks, no pressure.
- Not a social network — no likes count, no follower metrics visible.
- Not a news feed — curated, not infinite scroll.
