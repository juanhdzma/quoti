# Quoti — Color Palette

## Colors

| Token         | Hex       | Usage                                      |
|---------------|-----------|--------------------------------------------|
| `paper`       | `#FDFAF5` | Main background                            |
| `cream`       | `#F0E9DC` | Cards, secondary surfaces                  |
| `ink`         | `#1A1A1A` | Primary text, featured card background     |
| `stone`       | `#8A8278` | Secondary text, inactive icons             |
| `terracotta`  | `#C4604A` | Primary accent — CTAs, highlights, active  |
| `terra-soft`  | `#F5EAE7` | Accent tag backgrounds, soft highlights    |
| `ruled`       | `#E4DDD2` | Borders, dividers, muted lines             |

## Usage rules

- `paper` is always the screen background — never use pure white.
- `terracotta` is the single personality color — use it sparingly for maximum impact.
- `ink` on `paper` is the primary text pairing — never mix cold grays.
- `stone` replaces gray for secondary content — keeps warmth consistent.
- `terra-soft` is the only tinted surface for accent elements (tags, badges).
- Avoid adding new accent colors — extend with opacity variants if needed.
